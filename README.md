# FUTURE_UX_03
1. Overview
A CRM dashboard designed for service-based businesses to manage leads, clients, and tasks efficiently in one unified system.

2. Problem
Managers struggled to track deals, client interactions, and follow-ups across multiple tools, leading to missed opportunities and poor visibility.

3. Solution
Built a centralized dashboard with a visual pipeline, detailed client profiles, and an integrated task system to streamline workflows and improve decision-making.

4. Key Features
- Dashboard with revenue insights, alerts, and pipeline overview
- Visual lead pipeline with stage-based tracking
- Client management with CLV and detailed profiles
- Task tracker with follow-ups and priority grouping
- Team activity and performance snapshot
